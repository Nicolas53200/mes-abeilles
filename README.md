# 🐝 Mes Abeilles

**Carnet de rucher numérique** — Application web progressive (PWA)

## Fonctionnalités

- 🏠 Gestion des ruches et ruchettes avec fiches détaillées
- 📅 Enregistrement des visites (reine, couvain, varroa, hausses…)
- 🛡️ Suivi des traitements varroa et frelons
- 🍯 Historique des récoltes de miel
- 📋 Conformité apiculteur (NAPI, registre sanitaire, documents)
- ⌗ Génération de QR codes pour chaque ruche
- 📍 Géolocalisation GPS des ruchers
- 💾 Sauvegarde locale + export/import JSON
- 🪣 Gestion des ruchettes avec conversion en ruche

## Installation sur téléphone

### Android (Chrome)
1. Ouvre l'app dans Chrome
2. Menu ⋮ → **"Ajouter à l'écran d'accueil"**

### iPhone (Safari)
1. Ouvre l'app dans **Safari**
2. Bouton Partager ⬆️ → **"Sur l'écran d'accueil"**

## Technologies

- HTML / CSS / JavaScript vanilla
- PWA (manifest + service worker)
- localStorage pour la persistance des données
- Aucune dépendance externe
